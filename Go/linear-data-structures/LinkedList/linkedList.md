
### Linked List

We use a **linkedlist** to store objects in a squential manner in memory,       
But unlike arrays, likendlists can grow and shrink automatically.
    
/*     ---------    ---------           */
/*     | Value |    | Value |           */
/*     |  ---  |    |  ---  |           */
/*     |  next-|--> |  next-|--> NULL   */
/*     ---------    ---------           */
